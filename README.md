# Calidad-de-Software-Final
Proyecto 'Mentes Creativas': Frontend educativo (React/Vite) para 4º-5º grado (Matemáticas, Ciencias, Lógica). Enfocado en calidad ISO/IEC 25010(Usabilidad-Aprendizabilidad, Accesibilidad). El repositorio NO incluye código, solo evidencia de Plan de Calidad, Métricas, Estrategia de Pruebas, y la implementación de CI/CD y Despliegue.
